# Кластеризация объектов (KMeans)

Код: [`src/clustering.py`](../../src/clustering.py) и секция кластеризации в [`notebooks/baseline-metro_full.ipynb`](../../notebooks/baseline-metro_full.ipynb). Результат: `eda_output/final_clusters.csv`.

## Идея метода

Сгруппировать ~1500 объектов по **форме суточного профиля** (а не по абсолютному объёму), чтобы получить компактный категориальный признак «тип объекта». Кластер используется: как статический признак узла в GAT/глубоких моделях, для кластерного warp в синтетике и для иерархии станция→кластер→итог.

## Как работает (высокий уровень)

1. **Признаки профиля** (`build_profiles`): нормированный суточный профиль будней (24 доли) + сводные признаки — `morn_eve_ratio`, `we_wd_ratio`, `peakiness`, `night_share`, `midday_share`, `log_volume`.
2. **Фильтрация** слабых/вырожденных объектов (по объёму, пиковости).
3. **StandardScaler → KMeans** при разных `k` (4–14); выбор `k` по силуэту с ограничениями на размер/долю кластеров.
4. **Слияние** слишком мелких кластеров с ближайшими; присвоение человекочитаемых имён (например, «Жилой крупный (Метро)»).

## Применимость к малым данным

**Подходит.** Кластеризация ведётся по **усреднённым профилям** (агрегатам за неделю), а не по сырым почасовым точкам, поэтому устойчива к шуму малой выборки. Она снижает размерность «идентичности» объекта до нескольких групп — это регуляризует downstream-модели (общий сигнал на кластер вместо переобучения на отдельном объекте).

## Критичные гиперпараметры

- Диапазон `k` (4–14) и критерий отбора (силуэт + `max_share < 0.40`, `min_cl ≥ 10`).
- Фильтры объектов: `daily_volume ≥ 50`, `peakiness < 15`, `we_wd_ratio > 0`.
- Набор профильных признаков (24 доли + 6 сводных).
- `random_state=42`, `n_init` (стабильность KMeans).

## Ограничения

- Профили оценены на 1 неделе → возможна нестабильность кластеров при иной выборке.
- KMeans предполагает сферические кластеры в стандартизованном пространстве — грубое приближение.
- Имена кластеров эвристические (пороги по `morn_eve_ratio`/объёму), носят описательный характер.
